## Создание фиксации
Для создания фиксации используется команда *git commit*. Для этого в терминале с папкой-репозиторием необходимо написать команду *git commit -m <сообщение к комиту>*. Сообщение к коммиту писать **ОБЯЗАТЕЛЬНО**.
